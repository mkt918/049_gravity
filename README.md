# Gravity Lander - 重力着陸シミュレーター

太陽系の惑星に宇宙船を着陸させる2D物理シミュレーションゲームです。

## 🎮 ゲームの特徴

- **リアルな重力物理**: ニュートン力学に基づいたN体シミュレーション
- **時間ワープ機能**: 1x〜10000xまで時間を加速して長距離移動を短縮
- **精密な着陸チャレンジ**: 水星への着陸は特に難易度が高い！
- **美しいビジュアル**: サイバーパンク風の宇宙空間デザイン

## 🕹️ 操作方法

- **← →** : 宇宙船を回転
- **↑ / Space** : 推力を発射
- **T** : 時間倍率を切り替え（1x → 10x → 100x → 1000x → 10000x）
- **R** : ゲームをリセット
- **Z / X** : ズームイン/アウト

## 🚀 ローカルで実行

```bash
# 依存関係をインストール

# 開発サーバーを起動

```

ブラウザで `http://localhost:8080` にアクセスしてください。

## 🎯 ゲームの目標

地球の軌道から出発して、他の惑星（水星、金星、火星）に安全に着陸させることが目標です。

### 着陸成功の条件

- 着陸速度が5 m/s以下
- 惑星表面に接触

### 難易度

- **水星**: 最も難しい（太陽に近く、重力が複雑）
- **金星**: 中程度
- **火星**: 比較的簡単
